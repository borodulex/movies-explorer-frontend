## Как использовать

### Запуск сборки

1. Клонируйте репозиторий
2. В папке проекта выполните `yarn install` и одну из команд:

- Запуск в режиме разработки: `yarn start`;
- Сборка приложения в директорию build: `yarn build`.

3. Все, вы великолепны :)
